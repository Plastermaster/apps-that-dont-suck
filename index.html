Programs

Italics – paid software
Bold – “essential” recommendations

Drivers & Miscellaneous

  • Java 7 JRE / JDK
  • Adobe Flash Player – not recommended as it will be soon replaced with HTML5
  • Adobe Reader / Adobe Reader XI
  • Realtek High Definition Audio – i recommend finding a version for your motherboard first

File Sharing

  • Puush – upload and share files and screenshots easily with keyboard shortcuts
  • ShareX – open-source alternative to Puush
  • [Google Drive][google drive]
  • [Dropbox][dropbox]

File Compression

  • 7zip

Anti-Virus & Firewall

  • Comodo Internet Security
  • Malwarebytes
  • Kasperky

Social

  • Skype

Internet & Network

  • Firefox
  • Google Chrome
  • Fiddler – a web debugging proxy

Office

  • LibreOffice
  • OpenOffice
  • Microsoft Office

Text Editing

(These text editors are mostly targeted toward programmers.)

  • Atom
  • Notepad++
  • Sublime Text

Audio

  • foobar2000
  • Virtual Audio Cable
  • VB-Audio Virtual Cable

Audio Editing

  • Audacity

Media

  • VLC Media Player

Media Conversion

  • ffmpeg
  • FormatFactory

Game Capture

  • Dxtory
  • Fraps

Joypad-related Stuff

  • Joy2Key – translates joypad input to keyboard & mouse
  • Xpadder – alternative to Joy2Key
  • DS4Windows – allows you to use your ps4 controller as an xbox controller via usb or bluetooth

Music Production

  • FL Studio

Image Editing

  • Adobe Photoshop
  • Adobe Illustrator
  • Gimp
  • Inkscape

Live-Streaming

  • Open Broadcaster Software
  • XSplit

Looks & Themes

  • f.lux – colors your desktop depending on time of day
  • Rainmeter – add widgets, clocks, and that kind of crap to your screen
  • UXStyle – enables complete theming of your windows system, down to UI elements

Cleaning, Organization, Statistics, and Whatever

  • CCleaner
  • clover – adds tabs to your file explorer
  • Speccy – lists all your hardware specifications for easy bragging
  • Space Sniffer – visualizes locations on your hard drive as boxes depending on size
  • WinDivvy – precise window tiling

Programming

  • Git-SCM – a popular version control system
  • Microsoft Visual Studio – IDE for C++, C#, and VB.net
  • Intellij IDEA – IDE for Java and Javascript
  • Eclipse – another IDE for Java

Torrenting

  • UTorrent
  • BitTorrent
  • QBitTorrent

Websites

  • http://www.speedtest.net/ – show off how bad/good your internet is
  • http://10fastfingers.com/ – amazing typing test competition 2015
  • https://www.humblebundle.com/ – bundles of games for cheap and help charity

  • http://swfchan.net/ – a database of flash animations; totally not for porn (no seriously, don’t go here at work)

  • https://github.com/ – for git repositories, contribute to open source projects and whatever

  • https://about.gitlab.com/ – same as above, just an alternative

  • http://www.hitbox.tv/ – an alternative to twitch, personally experienced less stream delay

  • http://www.twitch.tv/ – who the hell doesn’t know about twitch

  • http://www.codecademy.com/ – learn to code (mostly in scripting languages tho)
  • http://regexr.com/ – play around with regular expression
  • http://choosealicense.com/licenses/ – know what license to choose for your open source project
  • https://open.gl/ – learn how to use opengl

  • http://www.ideacolorthemes.org/home/ – some cool themes if you use IDEA

  • http://mycolorscreen.com/ – a collection of smartphone themes for android and ios, also check out the android app “themer”

  • https://www.coinbase.com/ – a bitcoin service (i’ve made an account but never used it)

  • http://www.deviantart.com/browse/all/customization/skins/windows/visualstyle/ – my go-to for visual styles (the themes of windows); if you use uxstyle you won’t have to patch system files
